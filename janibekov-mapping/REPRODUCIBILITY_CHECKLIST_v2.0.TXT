SECTION 2: ENVIRONMENT SPECIFICATION
================================================================================

[✓] CLASS GridWorld  (janibekov_experiment.py, line ~110)
  Grid       : 10 × 10, flat index state = x*10 + y
  States     : |S| = 100
  Actions    : |A| = 4  (UP=0, DOWN=1, LEFT=2, RIGHT=3)  — deterministic
  Start      : (x=5, y=9)  bottom-centre
  Goal-A     : x < 5,  y < 5  (top-left quadrant)
  Goal-B     : x >= 5, y < 5  (top-right quadrant)
  Boundaries : clipped to [0,9]×[0,9]  (no walls, no holes)
  Episode T  : 200 steps (fixed; no early termination)

  Reward:
    R1 = 1.0 if next cell ∈ Goal-A, else 0
    R2 = 1.0 if next cell ∈ Goal-B, else 0
    R  = λ·R1 + (1−λ)·R2 − 0.01  (time penalty every step)

  Mode tracking per episode:
    sa = steps where agent entered Goal-A quadrant
    sb = steps where agent entered Goal-B quadrant

================================================================================
SECTION 3: Q-LEARNING AGENT
================================================================================

[✓] CLASS QAgent  (line ~145)
  Q-table   : list[list[float]], shape [100][4], init = 0.0
  α         : 0.10
  γ         : 0.99
  ε_explore : 0.05  (epsilon-greedy exploration)
  Update    : Q(s,a) ← Q(s,a) + α·[R + γ·max_a' Q(s',a') − Q(s,a)]

[✓] PERTURBATION  (function _gauss, line ~130)
  At action selection: perturbed_Q(s,a) = Q(s,a) + η_a
  η_a ~ N(0, ε²), independent per action per step
  Method: Box-Muller transform (pure stdlib, no numpy)
  ε = 0 during 100-episode pre-training phase.

[✓] PRE-TRAINING
  100 episodes, ε_noise=0, same λ as measurement phase.
  Seeds for pre-training: seed XOR 0xCAFE  (isolated from measurement rng).

SECTION 4: BEHAVIOR MODE DETECTION
================================================================================

[✓] FUNCTION episode_mode  (line ~175)
  frac_A = sa / (sa + sb)    [if sa+sb == 0, assign 'B']
  Mode-A : frac_A >= 0.60
  Mode-B : otherwise

[✓] FLIP DETECTION  (count_flips)
  flip_count = Σ 1[mode[i] ≠ mode[i−1]]  for i = 1..N_EP−1
  flip_rate  = flip_count / (N_EP − 1) × 100   [unit: per 100 episodes]

[✓] DWELL TIMES  (dwell_times)
  Run-length encoding of mode_sequence.
  Reported as mean ± SD across all dwell runs in the seed ensemble.

[✓] THRESHOLD SENSITIVITY
  0.60 verified at 0.50 and 0.70; qualitative results unchanged.

SECTION 5: STABILIZERS
================================================================================

[✓] INVARIANT CHECKER  (function checker_action, ~42 effective lines)
  Intercepts action selection. For each candidate action (ranked by Q-value):
    Simulate next cell (nx, ny).
    If current_mode=='A' and (nx,ny) ∈ Goal-B: SKIP.
    If current_mode=='B' and (nx,ny) ∈ Goal-A: SKIP.
    Otherwise: accept.
  Fallback: return highest-Q action if all actions violate.
  current_mode set at episode start from prior episode's label.
  First episode: current_mode = None → constraint inactive.

  Provenance: pure Python, written from scratch; no proprietary dependencies.

[✓] TMR VOTER  (class TMRVoter, ~78 effective lines)
  Three independent QAgents; seeds = [seed, seed+7, seed+13].
  Action = majority vote across 3 agents' select_action() calls.
  All three agents receive identical Q-learning updates each step.
  They diverge only via their independent perturbation samples.

  Provenance: generic triple-redundancy; no proprietary dependencies.

[✓] COMBINED  (function combined_action)
  Step 1: TMR majority vote → candidate action ta.
  Step 2: Invariant Checker filter on ta.
  Seeds for TMR in combined: [seed+1, seed+5, seed+11].

  TMR dwell           29.0          7.6           11.7              ✓ direction
  Combined dwell      89.3          120.0         800.0             ✓ direction

  R1 — TMR Suppression:
    The −81.2% figure in v1.0 was from a quick-mode run (N=200, 3 seeds).
    Full-N (N=1000, 20 seeds) gives −37.2%. The TMR mechanism is real and
    significant, but the benefit is smaller than initially reported.
    Paper Section 4.3 and Table 1 updated to −37.2%.

  R2 — Epsilon Scaling:
    The ε^0.74 claim in v1.0 was a log-log fit over only 3 seeds at N=200,
    which produced a spurious slope. Full-N data shows a plateau: flip rate
    ≈ 12.4–13.6 across ε ∈ {0.01, 0.05, 0.10, 0.20}. This is a stronger
    finding: the instability is geometric (landscape curvature at the
    intermediate axis), not noise-amplitude-driven. Paper Section 4.2
    revised to report the plateau and remove the power-law claim.
